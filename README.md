## Quick start
**At the root project level run the following commands:**
### To start Fast API backend run Uvicorn 
```
uvicorn main:app --reload
```
### To start Angular frontend run NPM
```
npm start
```
#### Navigate to localhost:8000

#### Open API Docs
http://localhost:8080/docs
